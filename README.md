## ‚ú® Features

- **Real-time Collaboration:** Type code and see changes appear instantly on other users' screens.
- **Room-based System:** Users can generate unique room IDs and invite others to join.
- **Syntax Highlighting:** Supports JavaScript, C++, Python, and Java via the Monaco Editor (VS Code core).
- **Instant Sync:** New users joining a room immediately receive the current code state (no more blank screens!).
- **User Notifications:** Get notified when users join or leave the room.
- **Responsive Design:** Works seamlessly on desktops, tablets, and mobile phones.
